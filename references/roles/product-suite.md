# Product Suite Roles

This reference defines the core product development roles within the squadron.

## 1. Architect (The Strategist)
- **Focus**: System design, scalability, and technical standards.
- **Workflow**: Drafts blueprints, chooses tech stack, and ensures cross-ship compatibility.
- **Tools**: Nelson's "Admiral" layer for planning.

## 2. Designer (The Polish)
- **Focus**: UX/UI, accessibility, and visual aesthetics.
- **Workflow**: Creates CSS variables, component templates, and interactive prototypes.
- **Standard**: Modern, responsive, and platform-native feel.

## 3. Full-stack (The PWO)
- **Focus**: Implementation across Frontend and Backend.
- **Workflow**: Fast implementation using Caveman-speak for speed.
- **Duty**: Write clean, idiomatic code that passes all Captain's inspections.

## 4. Mobile (Native/Hybrid)
- **Focus**: iOS, Android, and cross-platform (Flutter/Compose).
- **Workflow**: Native primitive usage and mobile-first optimization.
- **Duty**: Ensure "Hull Integrity" on mobile devices (battery, memory, data usage).
