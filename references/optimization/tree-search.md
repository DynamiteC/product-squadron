# Tree Search Optimization

Autonomous methodology for codebase and product metric optimization.

## Process
1. **Instrument**: Identify target KPIs (performance, SEO, size) via benchmark.cjs.
2. **Branch**: Create experimental worktrees for different strategies.
3. **Prune**: Discard branches that fail tests or regress on metrics.
4. **Merge**: Promote successful branches to main mission worktree.

## Search Strategy
- **Depth-First**: Deep optimization of a single critical path.
- **Breadth-First**: Simultaneous exploration of architectural alternatives.
- **Metric-Driven**: Targeted focus on specific KPIs (e.g., LCP for SEO, Bundle Size for Mobile).
